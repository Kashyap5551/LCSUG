Curriculum


  • Year 1 Core

    • EECS1001 - Research Directions in Computing

      • Research Directions in Computing

      This course is about some of the possible research topics in the realm of computer science. The topics and lectures vary every year. If you want to go into research in the futre, you'll enjoy this class.

      This is definitely the easiest course you'll ever take. It's pass or fail, so no grade is received. To pass, you just need to attend the classes and do the quizzes that are at the end of every lecture, with the occasional assignment strewn throughout the course.


      tl;dr - easiest course in the degree. You'll have a good time.


    • EECS1012 - Introduction to Computing: A Net-centric Approach

      • Introduction to Computing: A Net-centric Approach

      This is an introductory programming course. You'll learn the fundamentals, such as what variables are, beginner programming logic (conditional logic, loops, program tracing), and the basics of web development.

      You'll be working with HTML, CSS, and JavaScript in this course. If you did any programming on your own / in high school, this will be a walk in the park. If you didn't, you can still end with a high grade as long as you put in the time and effort needed (which is minimal). This course is definitely on the easier side of the spectrum.


      tl;dr - if you're a first time programmer then make sure you learn and understand basics of programming, else, this will be easy for you.


    • EECS1019 - Discrete Mathematics for Computer Science

      • Discrete Mathematics for Computer Science

      Discrete math is not like your average mathematics such as advanced functions or calculus. This is about the math that is related and necessary in computer science and includes topic like logic, Boolean algebra, set theory, and more.

      I found the first half of this course to be pretty easy (logic, Boolean algebra, and set theory) but it definitely ramped up when we started doing proofs, induction, counting principles, and more. This is a pretty abstract course and can be difficult to wrap your head around all the concepts, so some attention is needed.


      tl;dr - one of the more challenging first year courses with an easy first half with a harder second half. Don't take this with MATH1310.


    • EECS1022 - Programming for Mobile Computing

      • Programming for Mobile Computing

      A more ramped up version of EECS1012. Instead of web development, you'll be making mobile apps using Java and Android Studio. This would be a good time to mention that Java is the "main" language York teaches in; you'll have another 3-4 courses to take that uses Java.

      I found this course to be fun, although definitely more challenging. I remember having problems not only picking up Java for the first time, but also having to learn how to use the Android Studio IDE. It's important to leave this course knowing Object-Oriented Programming (OOP) concepts.


      tl;dr - if you've programmed before in any OOP language you'll be fine. Otherwise, take this course seriously and learn how to program in Java; we use it a lot.


    • MATH1300 - Differential Calculus with Applications

      • Differential Calculus with Applications

      This is Calculus I - mainly derivatives with some integrals near the end. If you did well in calculus in high school, you'll find majority of this course content to be similar, with the newer content not being that much more challenging.

      If you struggled with calculus in high school, you'll most likely struggle here too. Read the textbook; it really helped in my opinion. I also used this website a lot.


      tl;dr - good at calculus in high school? Good. No? Gonna have to work hard.


    • MATH1310 - Integral Calculus with Applications

      • Integral Calculus with Applications

      This is Calculus II - which is entirely about integrals. If you found calculus I to be easy, don't think that this course is anything like it. There is an insane ramp in difficulty between these two courses and a lot of difficult content. I recommend you pay a lot of attention and follow the textbook closely. This website helped a lot.


      tl;dr - I hope you're good with integrals. I would say this is one of the hardest first year courses, and recommend not doing this with EECS1019.


    • Foundational Science Requirement

      • Foundational Science Requirement

      You need to take 6.00 credits in sciences from a list of courses. Typically this means either two biology, two chemistry, one biology and one chemistry, or one physics course (biology and chemistry courses are 3.00 credits while physics is a 6.00 credit).

      I went with two chemistries (CHEM1000, CHEM1001) since I excelled in it in high school. Play to your strengths and what you did well in during high school.


      tl;dr - I can't speak for biology or physics, but I found the chemistry route to be enjoyable.


    • Electives

      • Electives

      You need 90 credits for the regular degree, and 120 credits for the honours degree. The computer Science, science, and mathematics requirement don't fill the requirement itself, so you'll be taking electives here and there. Generally, I advise against taking electives in first year unless you know you can handle it. I took my first year easy to gauge how I'd handle university life, and didn't take any electives.


      tl;dr - I advise against taking an elective in your first year since you should use it to gauge how difficult you find university and to set good habits, but electives I've taken and really enjoyed are MODR1770, GEOG2030, HUMA2220, and STS2411.
